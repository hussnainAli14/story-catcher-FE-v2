# Story Catcher V2 Frontend

This is the updated frontend for Story Catcher that connects to the new backend with dynamic GPT conversation flow.

## Features

- **Dynamic GPT Conversations**: No predefined templates - every question and reaction is generated by GPT
- **Empathetic Interviewer**: Compassionate, supportive, and encouraging tone
- **Real-time Storyboard**: Visual storyboard generation as you answer questions
- **Video Generation**: Automatic video creation from your story
- **Responsive Design**: Works on all devices

## Backend Connection

This frontend connects to: `https://story-catcher-backend-v2.onrender.com`

## Development

```bash
npm install
npm run dev
```

## Deployment

This project is configured for Vercel deployment.

## Key Differences from V1

- **No Templates**: All questions and reactions are dynamically generated by GPT
- **Personalized Flow**: Each conversation feels unique and connected
- **Enhanced Empathy**: More supportive and encouraging responses
- **Better Voiceover**: Video narration focuses on story content, not visual descriptions
